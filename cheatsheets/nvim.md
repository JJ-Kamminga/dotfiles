# Neovim cheatsheet

## Operators

d   delete
y   yank into register
gq  text formatting
>   shift right
<   shift left

## Word and text object motions

w   word forward
b   word backward
)   sentence forward
(   sentence backward
}   paragraph forward
{   paragraph backward
]   section forward
[   section backward

## Text object selection

Can be used in visual mode or after an operator.
